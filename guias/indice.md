# Guías

Esta sección reúne **material de soporte**. No forma parte del recorrido conceptual principal de `parte_1` a `parte_4`, pero acompaña la cursada en herramientas, instalación, escritura y trabajo cotidiano.

Sirve para resolver necesidades prácticas como:

- preparar el entorno,
- usar herramientas de terminal,
- trabajar con Git,
- escribir documentación técnica,
- generar diagramas.

:::{note}
Estas guías conviene consultarlas **cuando haga falta**, no necesariamente en orden lineal.
:::

## Qué tipo de material hay acá

| Grupo | Para qué sirve | Guías |
| :--- | :--- | :--- |
| Instalación y entorno | Preparar herramientas base de la cursada | [instalacion_jdk](./instalacion_jdk.md), [instalacion_intellij](./instalacion_intellij.md) |
| Herramientas de línea de comandos | Resolver tareas de trabajo cotidiano en terminal | [bash](./bash.md), [git](./git.md), [git_avanzado](./git_avanzado.md) |
| Documentación y diagramas | Escribir material técnico y producir apoyo visual | [markdown](./markdown.md), [guia_plantUML](./guia_plantUML.md) |
